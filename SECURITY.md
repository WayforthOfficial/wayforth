# Security Policy

## Reporting Vulnerabilities

Email: security@wayforth.io

Please do not open public GitHub issues for security vulnerabilities. Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (optional)

We will respond within 48 hours and aim to patch within 7 days.

## Smart Contract Security

Contracts are deployed on Base Sepolia testnet only.
Mainnet deployment follows an independent third-party security audit.
